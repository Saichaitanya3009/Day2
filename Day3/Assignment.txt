Question:Give me some idea on what is a Data type. What are Primitive Data types and Non-Primitive Data types. 
Explain with some examples?

Ans: A data type defines the kind of value a variable can hold in a programming language.

    1. Primitive Data Types:
        a. String
        b.Number
        c.Boolean
        d.Undefined
        e.Null
        f.Bigint
    
    2.Non-Primitive Data Types:
        a.Class
        b.Object
        c.Function
        d.Array
        e.Map


Examples for the primitive Data Types:

//String 

let str1 = "Chaithu";
console.log(str1 , typeof(str1));

let str2  = 'Sai';
console.log(str2 , typeof(str2));

let str3 = `chaitanay`;
console.log(str3 , typeof(str3));

//Boolean

let bool1 = true;
console.log(bool1, typeof(bool1));

let bool2 = false;
console.log(bool2 , typeof(bool2));

//number

let num1 = 10;
console.log(num1 , typeof(num1));
let num2 = 10.2345;
console.log(num2 , typeof(num2));
let num3 = 10e3;
console.log(num3 , typeof(num3));

//undefined

let a;
console.log(a , typeof(a));

//null

let x = null;
console.log(x , typeof(x));
